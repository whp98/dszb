# 主要设计思想
这个app主要是用于一些极客用于播放和分享自己的一些喜欢的直播源，来源可以是网络抓包，也可以是自建直播源，  
尽量避免被某些厂商所绑架。  
初期主要以可以获取的直播源为例子进行  
通过评分系统解决一些直播源失效问题，添加内容采取手动添加服务器模式，  
服务器选择功能主要适用于开源环境，用户可以自建服务器，用来搭建自己的专属服务，
可以丰富一些资源，和减少被和谐的风险
